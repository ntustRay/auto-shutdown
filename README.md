# Auto Shutdown System

A simple cross-platform Python script that automatically shuts down your computer at midnight (00:00) every day.

## Requirements

- Python 3.6 or higher
- schedule package

## Installation

1. Install the required package:

```bash
pip install -r requirements.txt
```

## Usage

1. Run the script:

```bash
python auto_shutdown.py
```

The program will run continuously and shut down your computer at midnight (00:00) every day.
To stop the program, press Ctrl+C.

## Supported Platforms

- Windows
- macOS
- Linux

## Note

Make sure to save all your work before the scheduled shutdown time, as the system will shut down automatically at midnight.
